## 支持的公式格式

### 行内公式
- 使用 `$...$` 包裹：`$x^2 + y^2 = 1$`
- 使用 `\(...\)` 包裹：`\(x^2 + y^2 = 1\)`

### 块级公式（独占一行，居中显示）
- 使用 `$$...$$` 包裹：
  ```
  $$\int_0^1 x^2 dx = \frac{1}{3}$$
  ```
- 使用 `\[...\]` 包裹：
  ```
  \[\sum_{i=1}^n i = \frac{n(n+1)}{2}\]
  ```

## 使用示例

### 在解析中使用公式

```
题目：求函数 $f(x) = x^2$ 在 $x=2$ 处的导数。

解析：
根据导数的定义：
$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

对于 $f(x) = x^2$：
$$f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h}$$
$$= \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h}$$
$$= \lim_{h \to 0} (2x + h) = 2x$$

所以当 $x=2$ 时，$f'(2) = 2 \times 2 = 4$
```

### 常用数学公式示例

#### 分数
- `$\frac{a}{b}$` → 显示为分数形式

#### 上下标
- `$x^2$` → x的平方
- `$x_i$` → x下标i
- `$x_i^2$` → x下标i的平方

#### 根号
- `$\sqrt{x}$` → x的平方根
- `$\sqrt[n]{x}$` → x的n次方根

#### 希腊字母
- `$\alpha, \beta, \gamma, \delta$` → α, β, γ, δ
- `$\pi, \theta, \sigma, \omega$` → π, θ, σ, ω

#### 运算符
- `$\times, \div, \pm, \cdot$` → ×, ÷, ±, ·
- `$\sum, \prod, \int$` → Σ, Π, ∫

#### 关系符
- `$\leq, \geq, \neq, \approx$` → ≤, ≥, ≠, ≈

#### 三角函数
- `$\sin(x), \cos(x), \tan(x)$` → sin, cos, tan

#### 极限
- `$\lim_{x \to \infty} f(x)$` → x趋向无穷时f(x)的极限

#### 矩阵
```
$$\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}$$
```

## 特性

1. **实时预览** - 解析和笔记字段输入时实时显示公式渲染效果
2. **自动识别** - 自动识别文本中的 LaTeX 公式并渲染
3. **错误处理** - 公式格式错误时显示原始文本，不影响其他内容
4. **响应式** - 块级公式支持横向滚动，适应小屏幕
5. **样式优化** - 公式字体大小、间距经过优化，阅读体验良好
6. **美观UI** - 预览区域带有渐变背景、圆角边框和阴影效果

## 编辑页面预览特性

### 解析字段预览
- 输入解析内容时，下方自动显示公式预览
- 预览区域带有"📐 公式预览"标题和"实时预览"标签
- 渐变背景色，视觉层次清晰
- 最大高度300px，内容过多时可滚动

### 笔记字段预览
- 与解析字段相同的预览体验
- 标题显示为"📐 笔记预览"
- 实时更新，方便核对公式格式

### MathInput组件
题目内容、正确答案、用户答案使用MathInput组件，提供：
- 公式插入工具栏（分数、根号、上下标、希腊字母等）
- 可切换的预览按钮（显示/隐藏预览）
- LaTeX语法提示

## 技术实现

- 使用 KaTeX 库（已安装在项目中）
- 支持 Vue 3 Composition API
- 使用计算属性自动响应内容变化
- v-html 安全渲染（公式由KaTeX生成，不含用户输入的HTML）

2. **AI解析测试**
   - 使用AI智能解析功能，查看解析结果中的公式渲染效果
   - 验证相关知识点和学习建议中的公式显示

   - 测试行内公式：`$x^2$`

## 注意事项

1. 确保 KaTeX CSS 已正确加载（已在组件中导入）
2. 块级公式前后最好有空行，以获得最佳显示效果
3. 如果公式显示异常，检查 LaTeX 语法是否正确
4. 复杂的公式可能需要使用 `\displaystyle` 来获得更好的显示效果
